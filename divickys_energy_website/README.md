# DIVICKYS Energy Resources Website

A functional Flask website built with Python, HTML, and CSS.

## Pages
- Home
- About
- Services
- Contact with form validation
- 404 error page

## How to Run

1. Install Flask:

```bash
pip install flask
```

2. Run the app:

```bash
python app.py
```

3. Open your browser and visit:

```text
http://127.0.0.1:5000
```
